## 项目说明
H5 Cloud 是一个基于 Vue 3 + Vant 开发的移动端 Web 应用。

### 技术栈
- Vue 3
- TypeScript
- Vant UI
- Antd Mobile
- TDesign Mobile Vue

### 开发环境要求
- Node.js >= 16
- MySQL >= 8.0
- Redis (可选)
